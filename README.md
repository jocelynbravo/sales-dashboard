# Sales & Revenue Dashboard

A Power BI–style interactive sales dashboard built with HTML, CSS, and Chart.js 

---

## Overview

This project simulates a real Power BI report environment, showcasing sales KPIs, regional breakdowns, and product performance for a fictional company (Contoso Corp) across fiscal year 2024.

It was built as a data analytics portfolio project to demonstrate dashboard design, data visualization, and front-end development skills 

---

## Features

- **Power BI dark theme** — custom dark UI matching the real Power BI aesthetic
- **4 KPI cards** — Total Revenue, Gross Profit, Total Orders, and Return Rate with YoY deltas
- **Interactive slicers** — filter by Year, Quarter, Region, and Category
- **Monthly Revenue vs Target** — combo bar + line chart with live filtering by quarter
- **Revenue by Region** — animated donut chart with regional breakdown
- **Revenue by Category** — horizontal bar chart (Software, Hardware, Services)
- **Top 5 Products table** — ranked with inline progress bars and YoY growth indicators
- **Fully responsive** — adapts to mobile and tablet screens
